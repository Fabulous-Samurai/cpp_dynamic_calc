# Contributing to AXIOM Engine

Thanks for contributing! Please follow these steps:

1. Fork the repository and create a feature branch.
2. Run tests locally and ensure new features include unit tests.
3. Keep changes small and focused; update documentation and changelog.
4. Open a pull request to `master` with a clear description and linked issues.
5. Ensure CI checks pass before requesting review.

Please read the `CODE_OF_CONDUCT.md` as well.
